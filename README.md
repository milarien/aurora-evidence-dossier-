# Aurora Evidence Dossier

This repository preserves the dated authorship, precedence, and evidence trail for the Aurora / PEF / Lens / Governor work.

It is an evidence archive, not an active product-code repository.

## Purpose

This dossier exists to document:

- the core architectural claim
- the identified failure class
- the supporting empirical and exhibit evidence
- the structural response developed in Aurora / PEF

The central claim is that dangerous LLM failure often appears not as simple false output, but as **continuation without warrant**: systems proceeding as though a state, fact, authority, or transition is settled when it is not.
