# Aurora-Lens Public Proof

This is a public proof and category-boundary surface for Aurora-Lens, not the production repository.

## Purpose

This package gives reviewers a concise public map of Aurora-Lens as a runtime admissibility and forensic governance system.

It exists to prevent category collapse. Aurora-Lens should not be reduced to a chatbot wrapper, dashboard, policy checklist, content filter, memory layer, or post-hoc audit tool.

For the evidence and precedence archive, see the [Aurora Evidence Dossier](https://github.com/milarien/aurora-evidence-dossier).
